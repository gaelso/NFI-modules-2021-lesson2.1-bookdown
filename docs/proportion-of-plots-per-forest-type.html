Proportion of plots per forest type

Let’s compare how many plots fell in each land cover class of Louland between systematic and simple random sampling (Table 4.1). We can see that our two simple sampling methods gave similar plot allocation per land cover. There is plus or minus 2 points between their proportion and the area proportion of the land cover classes.

Let’s refer back to the NFI eLearning Module 3: Sampling. The differences between these sampling methods do not translate to major differences in plot allocation per land cover classes, but rather to statistical considerations.
